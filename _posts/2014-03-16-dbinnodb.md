This command convert all tables to InnoDB


#### Config file

Configuration file : /path/to/config-file.json

```json
{
    "project": "Project name",

    "database": {
        "host": "127.0.0.1",
        "database": "wp_v381",
        "username": "root",
        "password": "----",
        "prefix": "wp_"
    }
}
```


#### Command

    appcli db:innodb /path/to/config-file.json
